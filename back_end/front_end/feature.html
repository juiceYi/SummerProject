VERRIFICATION CODE RECOGNITION / FEATURE

    欧码云验证码识别平台是由国内七人团队共同研发的网站，致力于解决自动验证码识别的问题

    全自动区分计算机和人类的公开图灵测试（CAPTCHA），俗称验证码，是一种区分用户是计算机或人的公共全自动程序。详情见 wiki

    本网站通过深度学习的方式识别验证码，而不是光学字符识别，请仅用于学习用途。阅读 免责声明

    欧码云平台的上传验证码识别完全免费，用户既可以上传识别，也可以由系统自动生成之。

    由于模型的识别准确率不是100%，所以希望您在发现验证码错误时，能把正确的结果输入进去，这对我们改善模型有帮助。
